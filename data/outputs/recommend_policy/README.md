# UI 전달용 데이터 설명

이 폴더에는 **UI 구현에 바로 사용할 수 있는 최소·핵심 데이터만** 포함되어 있습니다.  
(구 클릭 → 원인 설명 → 정책 방향 제시)

---

## 1️⃣ need_policy_recommendation_by_district.csv  ⭐ 핵심

### 용도
- **구 클릭 시 바로 보여줄 정책 방향 텍스트**
- “왜 이 구가 취약한지 + 무엇을 우선 개선해야 하는지” 설명용

### 주요 컬럼
- `district`  
  구 이름
- `top1_factor`, `top2_factor`, `top3_factor`  
  해당 구의 Need Index를 가장 크게 만드는 위험 요인 TOP3
- `policy_direction_1~3`  
  TOP 위험 요인 기반으로 자동 생성된 정책 방향 문장

### UI 사용 예
- 구 상세 카드:
  - **핵심 원인 TOP3**
  - **권장 정책 방향 2~3줄**

---

## 2️⃣ need_driver_top3_by_district.csv

### 용도
- **정책 방향의 수치적 근거 제시**
- 차트(막대/도넛) 또는 보조 설명용

### 주요 컬럼
- `district`  
  구 이름
- `need_factor`  
  위험 요인 변수명
- `contrib_score`  
  Need Index에 기여한 점수 (가중치 반영)
- `contrib_share_pct`  
  해당 요인이 Need Index에서 차지하는 비율(%)

### UI 사용 예
- “TOP3 원인 비중 그래프”
- “1위 원인 + %” 한 줄 요약

---

## 분석 개요 (참고)

- Need Index는 **정신건강·경제·고립·인구 구조 요인**을 종합한 지수입니다.
- 가중치는 교수님 자문을 반영하여 설정되었습니다.
- 각 구별로 Need Index를 구성하는 **주요 원인을 분해 분석**한 결과입니다.
- 본 데이터는 **진단 목적이 아닌 정책 우선순위 도출**을 위한 참고 자료입니다.
